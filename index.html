Descargar mpv

  1. abrir en navegador mpv.io
  2. click en installation (arriba en barra de navegación)
  3. en la sección "Windows" click en el primer link
  4. click botón verde (Download Latest Version)

Descargar NodeJS

  1. abrir en navegador nodejs.org
  2. click en botón verde izquierdo (Recommended for Most Users)
  3. guardar archivo (node-vxx.xx.x-x64.msi)

Permitir ejecución de scripts

  1. buscar "powershell" en buscador de Windows
  2. click derecho y seleccionar abrir como administrador
  3. pegar el siguiente texto: Set-ExecutionPolicy Bypass -Scope Process

Extraer mpv

  1. click derecho en archivo descargado de paso 1 (bootstraper.zip)
  2. seleccionar "Extraer todo"
  3. renombrar carpeta a "mpv"
  4. arrastrar carpeta a escritorio

Instalar mpv

  1. abrir carpeta mpv
  2. seleccionar updater.bat, click derecho y seleccionar abrir como administrador
  3. ESPERAR a que se instale (NO PRESIONAR NINGUNA TECLA)

Crear acceso directo

  1. abrir carpeta mpv
  2. click derecho en mpv y copiar ruta de accesso
  3. en escritorio click derecho, seleccionar nuevo -> accesso directo

Crear carpeta scripts

  1. presionar tecla Windows + r
  2. en el campo de entrada poner %appdata%
  3. click derecho en carpeta mpv, seleccionar nuevo -> carpeta
  4. renombrar nueva carpeta a "scripts"

Instalar NodeJS

  1. abrir instalador de nodejs descargado en paso 2
  2. selecctionar siguiente a todo y esperar instalación

Instalar script

  1. abrir powershell
  2. comprobar si npm está instalado escribiendo npm --version y presionando Enter. Si se muestra número de la forma X.X.X se puede proceder a instalar el script
  3. dentro de powershell pegar el siguiente texto npm install --global webtorrent-mpv-hook y esperar instalación
  4. dentro de powershell pegar el siguiente texto webtorrent-mpv-hook
  5. copiar la línea que empieza con "New-Item..." (seleccionando la línea y presionando ctrl+c)
  6. pegar ese texto en powershell y presionar Enter

Descargar un torrent

  1. abrir en navegador rarbgcore.org
  2. seleccionar una categoría en la barra de navegación izquierda (por ejemplo "Movies")
  3. ingresar CAPTCHA si es pedido
  4. seleccionar o buscar una película
  5. descargar el archivo torrent presionando el link arribar del póster

Abrir un torrent

  1. abrir mpv.exe desde el acceso directo
  2. arrastar el archivo torrent a la ventana de mpv
  3. esperar a que cargue la película
